Map Link - Desafio-dev-senior

Entrada

O contrato do componente define como entrada 
os seguintes parâmetros:
Lista de endereços
Cada endereço contém os dados:
- Nome da rua/avenida;
- Número;
- Cidade;
- Estado.
- Tipos de rota
Considere a relação de tipos de rota permitidos:
- Rota padrão mais rápida;
- Rota evitando o trânsito.

Saida

Tempo total da rota;
Distância total;
Custo de combustível;
Custo total considerando pedágio.

Proposta: plugin jquery



User interface enabled

$('#mlroutecost1').routecosts({
	addresses: [{state:'SP',city:'São Paulo',street:'Av. Paulista',number:'1000'}],
	routeType: 0,// default route type //http://dev.maplink.com.br/javascript-api/guia-referencia
	showButton: true, // display the button to toggle it
	showForm: true// // display the input form
});
